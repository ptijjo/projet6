# projet6 - API piquante

Prérequis :
Vous aurez besoin d'installer node et npm sur votre machine.


Installation de l'API :
Une fois le fichier téléchargé, ouvrez votre terminal de commande / IDE et se mettre dans le dossier : cd nom_fichier puis tapez npm install afin d'installer tous les fichiers de  node-modules.

Connection de l'API à votre base de donnée mongoDB :
Il faut se rendre sur env/urlConnection.js et y changer la valeur de urlConnection par les paramètres de connection de votre base de données mongoDB.


Lancer l'API :
Depuis votre terminal de commande / IDE tapez : node server.

L'API s'exécutera sur le localhost avec le port 3000.

Vous recevrez le message : "Connection à mongoDB réussie !".

Si vous avez ce message : "Connexion à MongoDB échouée !", il faudra vérifier vos paramètres de connection à votre base de données.
